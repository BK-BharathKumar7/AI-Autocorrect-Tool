# AI Autocorrect Tool

## Overview

The AI Autocorrect Tool is a Natural Language Processing (NLP) application that automatically detects and corrects spelling and grammar mistakes in user-entered text.

The project provides a simple and interactive web interface built using Streamlit and uses LanguageTool for text correction.

## Features

* Spelling correction
* Grammar correction
* Error detection
* Correction suggestions
* Download corrected text
* User-friendly interface

## Technologies Used

* Python
* Streamlit
* LanguageTool
* Natural Language Processing (NLP)

## Installation

Clone the repository:

git clone https://github.com/YOUR_USERNAME/AI-Autocorrect-Tool.git

Navigate to the project folder:

cd AI-Autocorrect-Tool

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py

Input:
I am lernning python and machne lernning.
